# Dataset Information

## Dataset Used
- **Indian Sign Language Skeletal-point NumPy Array (MediaPipe)**
- Source: IEEE DataPort

## Dataset Description
This dataset contains pre-extracted skeletal keypoints obtained using MediaPipe.
The data is suitable for time-series modeling for sign language recognition.
